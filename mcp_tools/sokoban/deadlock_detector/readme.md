# Sokoban Deadlock Checker Helper

A **minimal wrapper** around the `DeadlockDetector` class from  
[AI-Driven-Sokoban-Solver](https://github.com/shiandrew/AI-Driven-Sokoban-Solver).  
It converts a human-readable ASCII board to the solver’s internal format and
returns whether the position is provably unsolvable.
